# 搜索组件


:::tip
* 当前仅使用于搜索组件和表格搜索组件结合，单独使用场景还未测试
:::

* 场景一： 常规:一行展示xl:1920情况下搜索条件展示5条，
* 场景二： 常规:一行展示lg:1200情况下搜索条件展示4条
* 场景三： 侧边栏有树形结构，xl:1920情况下搜索条件展示4条，
  * 场景三： 侧边栏有树形结构，lg:1200情况下搜索条件展示3条

### 组件内容

≥1920px 响应式栅格数或者栅格属性对象
≥1200px 响应式栅格数或者栅格属性对象
